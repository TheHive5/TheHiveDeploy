⚠️ TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

"npm: command not found"
→ Install Node.js from nodejs.org

"Blank white window"
→ Check Vite server at http://localhost:5173
→ Press F12 to see console errors

"IPC not working"
→ Make sure .env has OPENAI_API_KEY

"Port 5173 in use"
→ See INSTALLATION.md troubleshooting section

More help?
→ See INSTALLATION.md (full troubleshooting)
→ Check relevant .md file
→ Review console for error messages
